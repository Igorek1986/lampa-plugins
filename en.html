Plugins for Lampa / Lampac

Plugins for the Lampa media center

Content catalog, watch tracking, serial status badges and a safe settings reset. No build step — installed via a link.

Choose a plugin GitHub →

File variants

Each plugin comes in three variants. They are functionally identical — only the file contents differ. The main .js link is minified and recommended for installation.

VariantFileWhat's inside
Defaultnp.jsMinified — smallest size, recommended for installation
Litenp.lite.jsReadable code without comments and logs
Fullnp.full.jsFull source with comments and logs (for debugging)
