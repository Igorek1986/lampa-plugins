Плагины для Lampa / Lampac

Плагины для медиацентра Lampa

Каталог контента, трекинг просмотра, статусы сериалов и безопасный сброс настроек. Без сборки — устанавливаются по ссылке.

Выбрать плагин GitHub →

Варианты файлов

Каждый плагин доступен в трёх вариантах. Они функционально идентичны — отличается только содержимое файла. Основная ссылка .js минифицирована и рекомендуется для установки.

ВариантФайлЧто внутри
По умолчаниюnp.jsМинифицированный — минимальный размер, рекомендуется для установки
Litenp.lite.jsЧитаемый код без комментариев и логов
Fullnp.full.jsПолный исходник с комментариями и логами (для отладки)
