Плагины для Lampa / Lampac

Плагины для медиацентра Lampa

Каталог контента, трекинг просмотра, статусы сериалов и безопасный сброс настроек. Без сборки — устанавливаются по ссылке.

Выбрать плагин GitHub →

Варианты файлов

Каждый плагин доступен в трёх вариантах. Они функционально идентичны — отличается только содержимое файла. Для слабых устройств рекомендуется .min.

ВариантФайлЧто внутри
Обычныйnp.jsПолный код с комментариями и логами
Litenp.lite.jsТо же, но без комментариев и логов
Minnp.min.jsКак Lite, дополнительно минифицирован
