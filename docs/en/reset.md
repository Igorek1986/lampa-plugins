🔄 Reset Settings plugin for Lampa  
A full reset of Lampa settings while preserving the unique device ID!  

🔹 Key features:

Fast and safe reset of all Lampa settings  
Preservation of important data:  
- Unique device identifier (lampac_unic_id)  
- Automatic reload after reset  
- Intuitive button in settings with a reset icon  
- Action confirmation to avoid accidental use  

🔹 How does it work?

- A "Reset settings" button appears in the Lampa settings menu  
- On press, a confirmation dialog opens  
- After confirming, all settings are cleared except lampac_unic_id  
- Lampa reloads automatically  

🔹 Technical highlights:

- Minimal code with no extra dependencies  
- Multi-language support (Russian, English, Ukrainian)  
- Compatible with the latest Lampa versions  
- Safe execution — does not touch critically important data  

⚡ Perfect if:

- Lampa started behaving unstably  
- You need to restore default settings  
- You need to clear the cache and old configurations  

Install it and enjoy a clean reset without losing key data!
