🎬 Плагин NUMParser для Lampa  
Мощный источник актуального контента с удобной фильтрацией и гибкими настройками!

🔹 Основные возможности:

- Обширный каталог с фильмами, сериалами и мультфильмами  
- Автоматическая фильтрация просмотренного контента.  
- Скрывает фильмы/сериалы при прохождении настраиваемого прогресса. Требудется модуль [TimecodeUser](https://github.com/Igorek1986/lampa-plugins/tree/main/module/TimecodeUser)
- Гибкие настройки отображения категорий  
- Поддержка 4K и новинок кинопроката  
- Интеграция с историей просмотров Lampa  

🗂 Категории контента:

- Новинки кино (включая русские фильмы)  
- Сериалы и мультсериалы  
- Контент в 4K качестве  
- Аниме и мультфильмы  
- Фильмы по годам выпуска (с 1980 по текущий год)  
- Топовые подборки  

🆕 Новое в плагине:

- **Порядок категорий** — кнопка «Порядок категорий» с перетаскиванием: меняйте местами и скрывайте любые категории под себя.  
- **Подборки по жанрам, актёрам и режиссёрам** — автоматические коллекции (комедии, боевики, триллеры, ужасы, фантастика и др.), каждый раз перемешиваются по-новому.  
- **Возрастной рейтинг на карточках** — цветные метки 0+/6+/12+/16+/18+ в углу постера. Настройка «Возрастной рейтинг на карточках».  
- **Формат качества** — выбор отображения качества: полный (WEBDL 1080p, BDRip…) или упрощённый (SD / 720p / 1080p / 4K).  
- **Скрыть без рейтинга** — отдельный переключатель, чтобы прятать контент без возрастной маркировки.  
- **Привязка устройства по коду** — больше не нужно вручную копировать токен: показывается код активации, который вводится на сайте («Мои устройства → Привязать устройство»). Ручной ввод токена тоже доступен.  
- **Учёт просмотров для «Популярного»** — раздел «Популярное» формируется на основе ваших просмотров; данные отправляются автоматически, без привязки устройства.  

⚙️ Удобные настройки:

- Настройка порога просмотра (50-100%)  
- Включение/отключение и порядок категорий  
- Возможность скрывать просмотренное и контент без рейтинга  
- Изменение названия источника в меню  
- Формат отображения качества видео  
- Поддержка мультиязычности (через настройки TMDB)  

🔧 Технические особенности:

- Оптимизированная загрузка контента  
- Поддержка карточек эпизодов для сериалов  
- Интеграция с системой избранного Lampa  
- Автоматическое обновление списков  
- Плагин идеально подходит для тех, кто хочет всегда иметь доступ к актуальному контенту с удобной системой фильтрации и персонализацией отображения.
