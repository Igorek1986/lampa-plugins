📺 Плагин MyShows для [Lampac](https://github.com/immisterio/Lampac) и [Lampa](http://lampa.mx/)  
Автоматическая синхронизация просмотра сериалов и фильмов с MyShows.me  

![Пример работы плагина](https://raw.githubusercontent.com/Igorek1986/lampa-plugins/main/docs/MyShows.png)

🔹 Основные возможности

- ✅ Авторизация через логин/пароль (не требует ручного ввода токена)  
- ✅ Поддержка мультипрофильности - индивидуальные настройки для каждого профиля Lampa  
- ✅ Автоматическое обновление токена при истечении срока действия  
- ✅ Отображение прогресса просмотра сериала  
- ✅ Отображение Следующей непросмотренной серии  
- ✅ "Непросмотренные сериалы (MyShows)" отображаеются на Главной CUB/TMDB
- ✅ Гибкие условия добавления сериалов в "Смотрю":  
- Сразу при запуске первой серии.  
- После просмотра от 5% до 50% первой серии  

- ✅ Точная синхронизация просмотра с настройкой порога отметки (50%-100%)

- ✅ Интеллектуальных режим работы:
- Отмечает просмотренные серии и фильмы

🆕 Новое в плагине:

- **Настраиваемые значки на карточках** — три независимых переключателя:  
- «Прогресс эпизодов» — просмотрено / всего серий (например, 5/12)  
- «Осталось серий» — количество непросмотренных серий  
- «Следующий эпизод» — номер следующей серии (например, S01E05)  
- **Расположение значков** — два варианта оформления:  
- Вариант 1 — классический.  
- Вариант 2 — следующий эпизод слева внизу, прогресс справа внизу, остаток серий справа вверху, со скруглениями как у карточки.  
- **Метка «Следующая серия»** — теперь видна и в окнах «Торренты» / «Онлайн», обновляется после закрытия плеера.  
- **Сортировка сериалов** — 7 вариантов: по алфавиту, по прогрессу, по числу непросмотренных, по дате последнего эпизода (↑/↓), по дате первого непросмотренного (↑/↓).  
- **Календарь MyShows** — даты выхода эпизодов из MyShows прямо в разделе «Календарь».  
- **«Непросмотренные сериалы» на Главной** — отдельный переключатель показа секции на главной CUB/TMDB.  
- **Режим хранения на NP-сервере** — настройка «Использовать NP сервер»: данные о непросмотренных сериалах хранятся на сервере для быстрой загрузки на всех устройствах.  
- **Синхронизация настроек между устройствами (NMSync)** — значки, сортировка и прочие настройки применяются на всех ваших устройствах автоматически.  
- **Ручная синхронизация с Lampac** — кнопка «Синхронизировать» для разовой полной сверки просмотренного.  

⚙️ Настройки плагина  
- 🔐 Аторизация через прокси-сервер (логин/пароль не хранятся в открытом виде) [myshows_proxy](https://github.com/Igorek1986/myshows_proxy)  
- 🎚 Порог добавления в «Смотрю» и порог отметки просмотра  
- ⏳ Время жизни кеша (7–90 дней)  
- 🎛 Показ и расположение значков, сортировка, кнопки управления в карточках  


🔧 Технические особенности 
- Работает со встроенным плеером и внешним плеером (отдающим timecode)
- 📊 Интеграция с системой хранения прогресса Lampac  
- 🔍 Определение сериалов по IMDB ID 
- 💾 Локальное кэширование данных для уменьшения запросов к API.  Время жизни кеша (7-90 дней)  
- 🔄 Синхронизация просмотренных фильмов и сериалов с Lampac. Требудется модуль [TimecodeUser](https://github.com/Igorek1986/lampa-plugins/tree/main/module/TimecodeUser)

📌 Как это работает  
- При старте просмотра:  

- Определяет сериал по IMDB ID (Kinopoisk ID, Original Titile and year)  
- Проверяет условия добавления в "Смотрю"  
- Создает локальный mapping серий  
- Определяет фильм по Оригинальному названию и году выпуска. 

- По завершению просмотра:  
- Отслеживает прогресс по таймлайну  
- При достижении порога - отмечает просмотр
