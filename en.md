# Plugin Repository for Lampa / Lampac

[Русский](https://igorek1986.github.io/lampa-plugins/) · **English**

A set of plugins for the [Lampa](http://lampa.mx/) and [Lampac](https://github.com/immisterio/Lampac) media center. Installed via a link — no build step, no dependencies.

## 📦 Plugin file variants

Each plugin comes in three variants. They are **functionally identical** — only the file contents differ:

| Variant | Example file | What's inside |
|---------|--------------|---------------|
| **Regular** | `np.js` | Full source code with comments and debug logs |
| **Lite** | `np.lite.js` | Same code, but **with comments and logs removed** |
| **Min** | `np.min.js` | Same as Lite, **additionally minified** (smallest file size) |

Use any variant — the behavior is the same. For low-end devices or slow connections, `.min` is recommended. To get a specific variant, replace `.js` in the URL with `.lite.js` or `.min.js`.
